CSS的精灵图、字体图标和用户界面样式

发表于 2022-08-01 更新于 2022-08-02 分类于 CSS 阅读次数：
学习笔记：这是一篇CSS的精灵图、字体图标和用户界面样式的学习笔记

精灵图（重点）、字体图标

精灵图意义

  1. 为什么需要精灵图

    • 一个网页中往往会应用很多小的背景图像作为修饰，当网页中的图像过多时，服务器就会频繁地接收和发送请求图片，造成服务器请求压力过大，这将大大降低页面的加载速度。

  2. 使用精灵图目的

    • 为了有效地减少服务器接收和发送请求的次数，提高页面的加载速度，出现了CSS精灵技术（也称CSS Sprites、CSS雪碧）。

  3. 核心原理

    • 将网页中的一些小背景图像整合到一张大图中 ，这样服务器只需要一次请求就可以了。

精灵图（sprites）的使用

  1. 使用精灵图核心：

    1. 精灵技术主要针对于背景图片使用，就是把多个小背景图片整合到一张大图片中
    2. 这个大图片也称为sprites精灵图或者雪碧图
    3. 移动背景图片位置， 此时可以使用background-position
    4. 移动的距离就是这个目标图片的x和y坐标。注意网页中的坐标有所不同
    5. 因为一般情况下都是往上往左移动，所以数值是负值
    6. 使用精灵图的时候需要精确测量，每个小背景图片的大小和位置

  2. 使用精灵图核心总结：

    1. 精灵图主要针对于小的背景图片使用。
    2. 主要借助于背景位置来实现-background-position 。
    3. 一般情况下精灵图都是负值（千万注意网页中的坐标：x轴右边走是正值，左边走是负值，y轴同理）

字体图标

字体图标的特点

  1. 字体图标的优点
    1. 轻量级：一个图标字体要比一系列的图像要小。一旦字体加载了，图标就会马上渲染出来，减少了服务器请求
    2. 灵活性：本质其实是文字，可以很随意的改变颜色、产生阴影、透明效果、旋转等
    3. 兼容性：几乎支持所有的浏览器，请放心使用
  2. 注意： 字体图标不能替代精灵技术，只是对工作中图标部分技术的提升和优化。
  3. 总结：
    1. 如果遇到一些结构和样式比较简单的小图标，就用字体图标。
    2. 如果遇到一些结构和样式复杂一点的小图片，就用精灵图。

字体图标的下载

  1. icomoon字库
  2. 阿里iconfont字库

字体图标的使用与引入

  1. 把下载包里面的fonts文件夹放入页面根目录下

  2. 在CSS样式中全局声明字体：简单理解把这些字体文件通过css引入到我们页面中；要注意字体文件路径的问题

    • 文件路径问题：路径需要根据字体图标文件夹icomoon中的style.css文件调用font文件夹中四个文件的路径而改变

      @font-face {
    font-family: 'icomoon';
    src: url('fonts/icomoon.eot?8axf2s');
    src: url('fonts/icomoon.eot?8axf2s#iefix') format('embedded-opentype'),
        url('fonts/icomoon.ttf?8axf2s') format('truetype'),
        url('fonts/icomoon.woff?8axf2s') format('woff'),
        url('fonts/icomoon.svg?8axf2s#icomoon') format('svg');
    font-weight: normal;
    font-style: normal;
    font-display: block;
}

  3. html标签内添加字体图标

    <!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    @font-face {
      font-family: 'icomoon';
      src: url('fonts/icomoon.eot?8axf2s');
      src: url('fonts/icomoon.eot?8axf2s#iefix') format('embedded-opentype'),
        url('fonts/icomoon.ttf?8axf2s') format('truetype'),
        url('fonts/icomoon.woff?8axf2s') format('woff'),
        url('fonts/icomoon.svg?8axf2s#icomoon') format('svg');
      font-weight: normal;
      font-style: normal;
      font-display: block;
    }

    .one {
      /* font-family与上方导入的代码中的第一行需要保持相同 */
      font-family: 'icomoon';
      font-size: 50px;
      color: pink;
    }

    .two {
      /* font-family与上方导入的代码中的第一行需要保持相同 */
      font-family: 'icomoon';
      font-size: 50px;
      color: skyblue;
    }
  </style>
</head>

<body>
  <span class="one"></span>
  <span class="two"></span>
</body>
</html>

字体图标追加

  1. 将压缩包内的selection.json重新上传即可，然后选中自己想要的新的图标，重新下载压缩包，并替换原来的文件
  2. 步骤
    1. 打开icomoon字库网址
    2. 点击左上角import icons按钮，弹出对话框，找到selection.json文件，打开，点击重新加载YES
    3. 开始选择新的字体图标，选择完毕后点击右下角generate font按钮，点击右下角download按钮下载

通过类名调用字体图标（推荐）

  1. HTML页面中引入字体图标中css文件 — style.css
    • 注意修改style.css文件中的四个路径，修改方法见上
  2. 标签直接调用图标对象的类名即可（类名在style.css文件中标注，或直接打开dome.html文件查询）

CSS用户界面样式

鼠标样式cursor

  1. 设置或检索在对象上移动的鼠标指针采用何种系统预定义的光标形状

  2. 语法:

    li {
   cursor: pointer; 
  }

  3. 属性值

    属性值 描述
    default 小白 默认
    pointer 小手
    move 移动
    text 文本
    not-allowed 禁止

轮廓线outline

  1. 给表单添加 outline: 0; 或者outline: none; 样式之后，就可以去掉默认的蓝色边框。

  2. 语法

    input {
     outline: none; 
}

防止拖拽文本域resize

  1. 实际开发中，我们文本域右下角是不可以拖拽的。

  2. 语法

    textarea{ 
       resize: none;
}

vertical-align属性应用

  1. CSS的vertical-align属性使用场景：经常用于设置图片或者表单（行内块元素）和文字垂直对齐。

  2. 官方解释: 用于设置一个元素的垂直对齐方式，但是它只针对于行内元素或者行内块元素有效。

  3. 语法：

    vertical-align : baseline | top | middle | bottom
    值 描述
    baseline 默认，元素放置在父元素的基线上
    top 把元素的顶端与行中最高元素的顶端对齐
    middle 把此元素放置在父元素的中部
    bottom 把元素的顶端与行中最低的元素顶端对齐

解决图片底部默认空白缝隙问题

  1. bug：图片底侧会有一个空白缝隙，原因是行内块元素会和文字的基线对齐。

  2. 解决方法有两种：

    1. 给图片添加vertical-align:middle | top| bottom;（提倡使用的）
    2. 把图片转换为块级元素display: block;

溢出的文字省略号显示

单行文本溢出显示省略号

/*1. 先强制一行内显示文本*/
white-space: nowrap;  （ 默认 normal 自动换行）
   
/*2. 超出的部分隐藏*/
overflow: hidden;
   
/*3. 文字用省略号替代超出的部分*/
text-overflow: ellipsis;

多行文本溢出显示省略号（了解）

/*1. 超出的部分隐藏 */
overflow: hidden;

/*2. 文字用省略号替代超出的部分 */
text-overflow: ellipsis;

/* 3. 弹性伸缩盒子模型显示 */
display: -webkit-box;

/* 4. 限制在一个块元素显示的文本的行数 */
-webkit-line-clamp: 2;

/* 5. 设置或检索伸缩盒对象的子元素的排列方式 */
-webkit-box-orient: vertical;

常见布局技巧

margin负值运用

1.让每个盒子margin 往左侧移动 -1px 正好压住相邻盒子边框

2.鼠标经过某个盒子的时候，提高当前盒子的层级即可（如果没有有定位，则加相对定位（保留位置），如果有定位，则加z-index）

文字围绕浮动元素

  1. 文字会围绕图片显示，因为创造浮动的初衷就是做文字环绕效果的

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }
        .box {
            width: 270px;
            height: 70px;
            background-color: pink;
            margin: 10px auto;
            padding: 5px;
        }
        .pic {
            float: left;
            width: 120px;
            height: 70px;
            margin-right: 5px;
        }
        img {
            width: 100%;
            height: 100%;
        }
    </style>
</head>
<body>
    <div class="box">
        <div class="pic">
            <img src="../images/bear.jpg" alt="">
        </div>
        <p>你好，你好呀</p>
    </div>
</body>
</html>

CSS三角强化

<style>
    .box1 {
        width: 0;
        height: 0;
        
        /* 把上边框宽度调大 */
        /* border-top: 100px solid transparent;
        border-right: 50px solid skyblue; */
        /* 左边和下边的边框宽度设置为0 */
        /* border-bottom: 0 solid blue;
        border-left: 0 solid green; */
        
        /* 1.只保留右边的边框有颜色 */
        border-color: transparent red transparent transparent;
        /* 2. 样式都是solid */
        border-style: solid;
        /* 3. 上边框宽度要大， 右边框 宽度稍小， 其余的边框该为 0 */
        border-width: 100px 50px 0 0 ;
    }

</style>
</head>
<body>
    <div class="box1"></div>
</body>
