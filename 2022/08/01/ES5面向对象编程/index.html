ES5面向对象编程

发表于 2022-08-01 更新于 2022-08-02 分类于 ES5 阅读次数：
学习笔记：这是一篇ES5面向对象编程的学习笔记

面向对象编程介绍

两大编程思想

  1. 面向过程
  2. 面向对象

面向过程编程

  1. 面向过程就是分析出解决问题所需要的步骤，然后用函数把这些步骤一步一步实现，使用的时候再一个一个的依次调用

面向对象编程

  1. 面向对象是把事务分解成为一个个对象，然后由对象之间分工与合作
  2. 面向对象是以对象功能来划分问题，而不是步骤
  3. 在面向对象程序开发思想中，每一个对象都是功能中心，具有明确分工
  4. 面向对象编程具有灵活、代码可复用、容易维护和开发的优点，更适合多人合作的大型软件项目

面向对象特性

  1. 封装性
  2. 继承性
  3. 多态性

面向过程和面向对象的对比

  1. 面向过程
    1. 性能比面向对象高适合跟硬件联系很紧密的东西
    2. 缺点：没有面向对象易维护、易复用、易扩展
  2. 面向对象
    1. 优点：易维护、易复用、易扩展由于面向对象有封装、继承、多态性的特性，可以设计出低耦合的系统，使系统更加灵活、更加易于维护
    2. 缺点：性能低于面向过程

面向对象Tab栏案例

抽象对象：Tab对象

  1. 该对象具有切换功能 — 切换方法
  2. 该对象具有添加功能 — 添加方法
  3. 该对象具有删除功能 — 删除方法
  4. 该对象具有修改功能 — 修改方法
  5. 因为页面一经加载就需要给元素绑定点击事件，因此需要一个初始化功能 — 初始化函数

面向对象版tab栏切换 — 添加功能

  1. 点击 + 可以实现添加新的选项卡和内容
  2. 第一步：创建新的选项卡li和新的内容section
  3. 第二步：把创建的两个元素追加到对应的父元素中.
  4. 以前的做法：动态创建元素createElement，但是元素里面内容较多，需要innerHTML赋值在appendChild追加到父元素里面。
  5. 现在高级做法：利用insertAdjacentHTML()可以直接把字符串格式元素添加到父元素中
  6. appendChild不支持追加字符串的子元素，insertAdjacentHTML支持追加字符串的元素
  7. 因为每次进行tab栏添加后，都需要重新获取li和section，所以将获取li和section封装为一个函数，每次点击添加后都进行调用

面向对象版tab栏切换 — 删除功能

  1. 击’x’可以删除当前的li选项卡和当前的section
  2. ‘x’是没有索引号的，但是它的父亲li有索引号，这个索引号正是我们想要的索引号
  3. 所以核心思路是：点击’x’号可以删除这个索引号对应的li和section

面向对象版tab栏切换 — 编辑功能

  1. 双击选顶卡li或者section里面的文字，可以实现修改功能
  2. 双击事件是：ondblclick
  3. 如果双击文字，会默认选定文字，此时需要双击禁止选中文字
    1. 方法一：js中在对应方法中添加代码
      • window.getSelection?window.getSelection().removeAllRanges():document.selection.empty();
    2. 方法二：在对应li和section的css样式中添加，会导致单击也无法选中
      • user-select: none;
  4. 核心思路：双击文字的时候，在里面生成一个文本框，当失去焦点或者按下回车键时，把文本框输入的值给原先元素即可

  2. js代码

    // 创建一个tab类
// var that;
class Tab {
  constructor(id) {
    // that = this;
    // 获取实例化对象中的元素
    // 先获取整个tab
    this.main = document.querySelector(id);
    // 获取添加按钮
    this.appends = this.main.querySelector('.append');
    // 获取ul
    this.ul = this.main.querySelector('.tab-top ul:first-child');
    // 获取tab-content    
    this.tab_content = this.main.querySelector('.tab-content');
    // 实例化对象时即调用初始化函数
    this.init();
  }

  // 初始化函数
  init() {
    // 初始化时，获取li和section
    this.updateNode();
    // 给appends绑定点击事件
    // 这里调用函数不要加()，因为此处是点击后才调用，加括号后则是页面加载即调用
    this.appends.onclick = this.addTab.bind(this.appends, this);
    // 给lis绑定点击事件
    for (var i = 0; i < this.lis.length; i++) {
      // 给每一个li设置索引号
      this.lis[i].index = i;
      // 这里调用函数不要加()，因为此处是点击后才调用，加括号后则是页面加载即调用
      this.lis[i].onclick = this.toggleTab.bind(this.lis[i], this);
      // 给每一个span绑定点击事件
      // 这里调用函数不要加()，因为此处是点击后才调用，加括号后则是页面加载即调用
      this.spans[i].onclick = this.removeTab.bind(this.spans[i], this);
      // 给每一个li绑定双击事件
      this.lis[i].ondblclick = this.editTab;
      this.sections[i].ondblclick = this.editTab;
    }
  }

  // 因为经常进行tab栏的添加删除，所以要多次获取li、section和span
  updateNode() {
    // 获取所有的li
    this.lis = this.main.querySelectorAll('li');
    // 获取所有的section
    this.sections = this.main.querySelectorAll('section');
    // 获取所有的span
    this.spans = this.main.querySelectorAll('span');
  }

  // 切换功能
  toggleTab(that) {
    // tab-top实现切换功能
    // 这里的that为指向实例化对象
    that.clearClass();
    // 留下自己，给被点击的li添加类名
    this.className = 'backcolor';

    // tab-content实现切换功能
    // 留下自己，给被点击的li对应的section添加类名
    that.sections[this.index].className = 'block';
  }
  // 排他思想清除类函数
  clearClass() {
    // 因为这个函数是由that调用的，所以this指向的是实例化对象
    for (var j = 0; j < this.lis.length; j++) {
      // 排他思想：干掉别人，让所有的li去掉类
      this.lis[j].className = '';
      // 排他思想：干掉别人，清除每一个section的类
      this.sections[j].className = '';
    }
  }
  // 添加功能
  addTab(that) {
    // 创建一个li和span节点，将span节点放入li中
    // 创建li元素
    var li = '<li>新选项卡<span></span></li>';
    // 将li元素插入ul中
    that.ul.insertAdjacentHTML('beforeend', li);

    // 创建section元素，添加入tab-content中
    var section = '<section>请输入内容</section>';
    // 将section元素添加进入ab-content中
    that.tab_content.insertAdjacentHTML('beforeend', section);
    // 每次添加li和section元素时，都需要重新初始化，重新调用li和section
    that.init();
  }
  // 删除功能
  removeTab(that, e) {
    // 阻止冒泡，避免点击span'x'号后，冒泡触发点击li的事件
    e.stopPropagation();
    // 因为span没有索引，则获取当前被点击span的父元素的索引
    var index = this.parentNode.index;
    // 删除索引号对应的li和section
    that.lis[index].remove();
    that.sections[index].remove();
    // 每次删除li和section元素时，都需要重新初始化
    that.init();
    // 当删除的是未被选中的tab选项时，被选中的tab选项不变
    if (that.main.querySelector(".backcolor")) return;
    // 当删除被选中的tab选项时，该tab选项的前一个tab选项会被自动选中
    // 可以采用被自动点击的方式实现
    index--;
    // 当所有的tab选项被删除时，index为-1， that.lis[index]不存在则停止点击事件
    that.lis[index] && that.lis[index].click();
  }
  // 修改功能
  editTab() {
    // 取消默认双击选定文字
    window.getSelection ? window.getSelection().removeAllRanges() : document.selection.empty();
    // 获取li中的文字
    var str = this.innerText;
    // 每当li被双击时，就在span内部生成一个input表单替换span里面的文字
    this.innerHTML = '<input type="text">';
    // 获取对应的input
    var input = this.children[0];
    // 将li中的文字赋值给文本框
    input.value = str;
    // 让文本框中的内容处于选中状态
    input.select();

    // 当文本框失去焦点或者按下enter键则将文本框的内容赋值给li
    input.onblur = function () {
      // 这里的this指向input
      this.parentNode.innerHTML = input.value + '<span></span>';
    }
    // enter键的ASCII码值为13
    input.onkeyup = function (e) {
      if (e.keyCode === 13) {
        this.blur();
      }
    }
  }
}

// 实例化一个tab对象
new Tab('#tab');
