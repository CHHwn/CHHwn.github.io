JavaScript内置对象和数据类型

发表于 2022-08-01 更新于 2022-08-02 分类于 JavaScript 阅读次数：
学习笔记：这是一篇JavaScript内置对象和数据类型的学习笔记

内置对象

  1. JavaScript中对象分为三种：自定义对象，内置对象，浏览器对象
  2. 内置对象就是指JS语言自带的一些对象，这些对象供开发者使用，并提供了一些常用的或是最基本而必要的功能(属性和方法)

查文档

  1. 学习一个内置对象的使用，只要学会其常用成员的使用即可，可以通过查文档学习，可以通过MDN/W3C来查询
  2. MDN网址： https://developer.mozilla.org/zh-CN/

Math数学对象

  1. Math对象不是构造函数，它具有数学常数和函数的属性和方法；跟数学相关的运算(求绝对值，取整、最大值等)可以使用Math中的成员

  2. Math对象部分属性和方法

    属性或方法 描述
    math.PI 圆周率
    math.floor() 向下取整
    math.ceil() 向上取整
    math.round() 四舍五入版，就近取整（负数中.5特殊，往大取；如-1.5四舍五入得-1）
    math.abs() 绝对值
    math.max() 最大值
    math.min() 最小值

  3. math对象随机数方法

    • math.random();
    1. 返回一个随机的小数，范围[0, 1)
    2. 这个方法内不跟参数
    3. 想要得到两个数之间的随机整数并且包含这2个整数
      • return Math.floor(Math.random() * (max - min + 1)) + min;

Date日期对象

  1. Date日期对象：是一个构造函数，必须使用new来创建日期对象（实例化）

  2. Date实例用来处理日期和时间

  3. 使用Date对象

    1. 没有参数：返回当前系统的当前时间

      var date = new Date();

    2. 有参数：返回参数的时间；日期格式字符串为’2019-5-1’，可以写成new Date(‘2019-5-1’)或new Date(‘2019/5/1’)

      var date = new Date('2019-5-1');
var date = new Date('2019/5/1');

日期格式化

  1. 获取日期指定的部分，手动得到需要的格式

    方法名 说明
    getFullYear() 获取当年
    getMonth() 获取当月（0~11）
    getDate() 获取当天日期
    getDay() 获取星期几（周日0 到 周六6）
    getHours() 获取当前小时
    getMinutes() 获取当前分钟
    getSeconds() 获取当前秒钟

  2. 案例：格式化日期年月日星期

    <script>
  // 创建日期对象
  var date = new Date();

  var year = date.getFullYear();
  var month = date.getMonth();
  var dates = date.getDate();
  var day = date.getDay();
  var arr = ['星期日', '星期一', '星期二', '星期三', '星期四', '星期五', '星期六'];

  console.log('现在的时间是：' + year + '年' + (month + 1) + '月' + dates + '日' + ' ' + arr[day]);
</script>

获取总的毫秒数

  1. Date对象是基于1970年1月1日（世界标准时间）起的毫秒数（时间戳）

  2. 通过value Of()和getTime()两个函数都可以得到

    var date = new Date();
console.log(date.valueOf()); // 就是现在时间距离1970年1月1日总的毫秒数
console.log(date.getTime()); // 就是现在时间距离1970年1月1日总的毫秒数

  3. 简单写法（常用）

    var date = +new Date(); // +new Date() 返回的就是总的毫秒数

  4. H5新增获取总毫秒数

    console.log(Date.new());

  5. 将总毫秒数转换为天、时、分、秒（时间戳转换为时分秒）

    1. 转换公式如下：

      • 计算天数：parseInt(总秒数/60/60/24);
      • 计算小时：parseInt(总秒数/60/60%24);
      • 计算分钟：parseInt(总秒数/60%60);
      • 计算秒数：parseInt(总秒数%60);

    2. 案例：倒计时

      <script>
  function countTime(time) {
    // 到当前时间的总毫秒数
    var nowTime = +new Date();
    // 距离用户设置时间time的总毫秒数
    var inputTime = +new Date(time);
    // 现在距离设置时间所剩余的秒数
    var times = (inputTime - nowTime) / 1000;

    // 天
    var day = parseInt(times / 60 / 60 / 24);
    day = day < 10 ? '0' + day : day;
    // 时
    var hour = parseInt(times / 60 / 60 % 24);
    hour = hour < 10 ? '0' + hour : hour;
    // 分
    var minute = parseInt(times / 60 % 60);
    minute = minute < 10 ? '0' + minute : minute;
    // 秒
    var second = parseInt(times % 60);
    second = second < 10 ? '0' + second : second;

    return day + '天' + hour + '时' + minute + '分' + second + '秒';
  }
  console.log(countTime('2021-7-22 18:00:00'));
</script>

数组对象

检测是否为数组

  1. 方式一：instanceof — 运算符，用来检测是否为数组

    var arr = [];
var obj = {};
console.log(arr instanceof Array); // true
console.log(obj instanceof Array); // false

  2. 方式二：Array.isArray(参数); — H5新增的方法，ie9以上版本支持

    console.log(Array.isArray(arr)); // true
console.log(Array.isArray(obj)); // false

添加数组元素

  1. push(参数1..)：在数组的末尾添加一个或多个元素

    1. 语法

      var arr = [1, 2, 3];
arr.push(4, 'pink');
      • 数组元素直接写数组元素
      • push完毕以后，返回的结果是新数组的长度
      • 原数组也会发生变化

  2. unshift(参数1..)：在数组的开头，添加一个或者多个数组元素

    1. 语法arr.unshift('red', 'purple');
      • 数组元素直接写数组元素
      • unshift完毕以后，返回的结果是新数组的长度
      • 原数组会发生变化

删除数组元素

  1. pop()：删除数组的最后一个元素(内部无参数)
    1. 语法：arr.pop();
      • pop()没有参数
      • pop完毕以后，返回值是删除的元素
      • 原数组会发生变化
  2. shift()：删除数组的第一个元素(内部无参数)
    1. 语法：arr.shift();
      • shift()没有参数
      • shift完毕以后，返回值是删除的元素
      • 原数组会发生变化

数组翻转和排序

  1. reverse()：颠倒数组中元素的顺序（内部无参数）

    1. 语法：arr.reverse();

  2. sort()：对数组中的元素进行排序（内部无参数）

    1. 语法：

      arr.sort(function(a, b){
  return a -b; //升序排序
  return b - a; //降序排序
});

数组索引方法

  1. indexOf(数组元素, [开始查找位置的索引])：数组中查找给定元素的第一个索引（从前到后查找）
    1. 语法：arr.indexOf('blue');
      • 只返回第一个满足条件的索引号
      • 若数组内没有该元素，则返回-1
  2. lastIndexOf(数组元素)：数组中查找给定元素的最后一个索引（从后到前查找）
    1. 语法：arr.lastIndexOf('blue');
      • 只返回最后一个满足条件的索引号
      • 若数组内没有该元素，则返回-1

数组转换为字符串

  1. toString()：将数组中的元素转换为字符串
    1. 语法：arr.toString();
      • 返回值为数组转换后的字符串
  2. **join(‘分隔符’)**：将数组中的元素转换为字符串，可以改变连接的符号
    1. 语法：arr.join('-');

数组其他方法

  1. **concat()**：连接两个或者多个数组，不影响原数组

    1. 语法：arr1.concat(arr2);
      • 返回值为新的数组

  2. **slice(begin, end)**：数组截取，截取从数组第begin个索引到第end个索引的元素，不包含索引为end的元素

    1. 语法：arr.slice(2, 4);
      • 返回值为被截取项目的新数组
      • 截取元素中不包含索引为end的元素
    2. 若只有一个参数begin，则表示截取从begin开始往后的所有元素

  3. splice()：删除元素 / 插入元素 / 替换元素

    1. 语法：arr.splice(begin, count, value);

      • 删除元素：第一个参数为从第几个索引开始，第二个参数为删除多少个
      • 替换元素：第一个参数为从第几个索引开始，第二个参数为要替换几个元素，后面的参数为替换前面的元素
      • 插入元素：第一个参数为从第几个索引开始，第二个参数为0，后面的参数为要插入的元素

    2. 返回值为被删除项目的新数组，会影响原数组的元素

      // 数组中添加新元素
var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.splice(2,0,"Lemon","Kiwi"); // Banana,Orange,Lemon,Kiwi,Apple,Mango

// 移除数组的第三个元素，并在数组第三个位置添加新元素
var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.splice(2,1,"Lemon","Kiwi"); // Banana,Orange,Lemon,Kiwi,Mango

// 从第三个位置开始删除数组后的两个元素
var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.splice(2,2); // Banana,Orange

  4. 字符串所有的方法，都不会修改字符串本身（字符串是不可变的），操作完成会返回一个新的字符串

基本包装类型

  1. 基本包装类型：把简单数据类型，包装成为了复杂数据类型

  2. 对象和复杂数据类型才具有属性和方法

    var str = 'andy';
str.length;
    1. ‘andy’为字符数据类型，属于简单数据类型，为什么会有length属性？
      • 因为基本包装类型把简单数据类型包装为了复杂数据类型
        • 第一步：把简单数据类型包装为复杂数据类型，执行过程如下
          var temp = new String('andy');
        • 第二步：把临时变量的值给str
          str = temp;
        • 第三步：销毁临时变量
          temp = null;

根据字符返回位置

  1. indexOf(‘要查找的字符’, [开始查找位置的索引])：字符串中查找给定元素的第一个索引（从前到后查找）

    1. 语法：

      str.indexOf('a', 5);
      • 返回值为指定内容在原字符串中的位置，若找不到则返回-1

  2. lastIndexOf(‘要查找的字符’)：字符串中查找给定元素的最后一个索引（从后到前查找）

    1. 语法：

      str.lastIndexOf('b');
      • 返回值为指定内容在原字符串中的位置，若找不到则返回-1

根据位置返回字符（重点）

  1. charAt(index)：返回指定位置的字符（index为字符串的索引号）

    1. 语法：str.charAt(2);

  2. charCodeAT(index)：获取指定位置处字符的ASCII码（index为索引号）

    1. 语法：

      str.charCodeAt(3);
      • 目的是可以判断用户按下了哪个键

  3. str[index]：返回指定位置的字符（HTML5，IE8+支持）

    1. 语法：str[1];

  4. 案例：统计一个字符串中每个字符出现的次数，求出出现最多的字符及其出现次数

    • 以下算法很常用

      <script>
  var str = 'abcoefoxyozzopp';
  var statistical = {};
  for (var i = 0; i < str.length; i++) {
    var chars = str.charAt(i); // 遍历字符串内的每一个字符
    // statistical[chats]是一个属性值，当对象中没有该属性时，属性值为0
    if (statistical[chars]) {
      // 代表该字符已经在对象出现过，将属性值自增1       
      statistical[chars]++;
    } else {
      // 代表该字符在对象中第一次出现，让该属性的值为1
      statistical[chars] = 1;
    }
  }
  console.log(statistical);

  // 遍历对象
  var max = 0;
  var ch = '';
  for (var k in statistical) { // k为变量，输出为属性名
    if (statistical[k] > max) {
      max = statistical[k]; // statistical[k]输出为属性值
      ch = k;
    }
  }
  console.log('字符串中出现最多的字符为：' + ch + '，出现次数为' + max + '次');
</script>

字符串操作方法（重点）

  1. concat(str1, str2, str3..)：用于连接两个或多个字符串，拼接字符串，等效与+，但+更常用

    1. 语法

      str.concat(str1, str2, str3..)
str.concat('red');

  2. substr(start, length)：从start位置开始（索引号），length为取的个数

    1. 语法：

      str.substr(1, 3);
      • 返回值为被截取的字符

  3. slice(begin, end)：字符串截取，截取从字符串第begin个索引到第end个索引的字符，不包含索引为end的字符

    1. 语法：

      str.slice(2, 4);
      • 返回值为被截取的字符
      • 截取元素中不包含索引为end的字符

  4. substring(begin, end)：字符串截取，截取从字符串第begin个索引到第end个索引的字符，不包含索引为end的字符

    1. 语法：

      str.substring(2, 4);
      • 返回值为被截取的字符
      • 截取元素中不包含索引为end的字符
      • 基本和slice相同，但是不接受负值

  5. split(str, num)：通过指定分隔符对字符串进行切片，如果参数num有指定值，则分隔num+1个子字符串

    1. str：分隔符，默认为所有的空字符，包括空格、换行(\n)、制表符(\t)等

    2. num：分割次数。默认为 -1, 即分隔所有

    3. 注意：返回值为一个数组

      str = "Line1-abcdef \nLine2-abc \nLine4-abcd";
print str.split( );       # 以空格为分隔符，包含 \n
print str.split(' ', 1 ); # 以空格为分隔符，分隔成两个

// 以上输出如下
// ['Line1-abcdef', 'Line2-abc', 'Line4-abcd']
// ['Line1-abcdef', '\nLine2-abc \nLine4-abcd']

替换字符串

  1. replace(‘被替换的字符’, ‘替换为的字符’)：替换字符串中指定的字符

    1. 语法：

      str.replace('a', 'b');
      • 返回值为被替换后的字符串

    2. 案例：替换敏感词

      <script>
  var str = '内置对象就是指JS语言自带的一些对象，这些对象供开发者使用，并提供了一些常用的或是最基本而必要的功能(属性和方法)';

  while (str.indexOf('些') !== -1) {
    str = str.replace('些', '*');
  }
  console.log(str);
</script>

字符串转换为数组

  1. **split(‘分隔符’)**：将字符串中的字符转换为数组元素，分隔符为需要转换为数组的字符串所带的符号

    1. 语法

      var str = 'red-green-blue';
str.split('-');
      • 返回值为字符串转换后的数组

简单数据类型和复杂数据类型

  1. 简单类型又叫做基本数据类型或者值类型，复杂类型又叫做引用类型

简单数据类型

  1. 简单数据类型（值类型）：简单数据类型/基本数据类型，在存储时变量中存储的是值本身，因此叫做值类型
  2. 五大简单数据类型（值类型）：string, number, boolean, undefined, null

null

  1. 简单数据类型null，返回的是一个空对象 — object

  2. 若存在一个变量我们打算存储为对象，却暂时没有想好放什么，这时就可以给null

    var timer = null;
console.log(typeof timer);

栈

  1. 栈（操作系统）：由操作系统自动分配释放存放函数的参数值，局部变量的值等
  2. 简单数据类型存放到栈里面：存放的是值

简单数据类型传参

  1. 函数的形参可以看做是一个变量，当我们把一个值类型变量作为参数传给函数的形参时，其实是把变量在栈空间里的值复制了一份给形参，那么在方法内部对形参做任何修改，都不会影响到的外部变量

复杂数据类型

  1. 复杂数据类型（引用类型）：在存储是变量中存储的仅仅是地址（引用）
  2. 通过new关键字创建的对象（系统对象，自定义对象），如Object，Array，Date等

堆

  1. 堆（操作系统）：存储复杂数据类型（对象），一般由程序员分配释放，若程序员步释放，则由垃圾回收机制回收
  2. 复杂数据类型存放到堆里面：首先在栈里面存放地址（十六进制表示），然后这个地址指向堆里面的数据

复杂数据类型传参

  1. 函数的形参也可以看做是一个变量，当我们把引用类型变量传给形参时，实是把变量在栈空间里保存的堆地址复制给了形参，形参和实参其实保存的是同一个堆地址，所以操作的是同一个对象

  2. 案例

    <script>
  function Person(name) {
    // 将栈内存中地址为0xaabbcc指向堆内存中对象p的name属性值改为'刘德华'
    this.name = name;
  }

  function f1(x) { // 这里x在栈内存中的地址为0xaabbcc，指向了堆内存中的对象
    // 输出堆内存中对象的name属性值，因为地址一致，因此地址指向相同，为同一对象
    console.log(x.name); // 第二：输出刘德华
    // 将堆内存中对象的name属性值改为'张学友'
    x.name = '张学友';
    console.log(x.name); // 第三：输出张学友
  }

  // 创建对象p，将对象p的地址存储进栈内存，假设地址为0xaabbcc；地址指向堆内存中的对象，对象的name属性值为'刘德华'
  var p = new Person('刘德华');
  // 通过栈内存中的地址0xaabbcc，找到堆内存中p对象的name属性值
  console.log(p.name); // 第一：输出刘德华
  // 将栈内存中p对象的堆地址0xaabbcc复制给形参
  f1(p);
  // 此时栈内存中0xaabbcc地址指向的对象属性值已经被更改
  console.log(p.name); // 第四：输出张学友
</script>
